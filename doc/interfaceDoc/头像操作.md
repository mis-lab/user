# 头像操作接口文档

### 1.头像上传

##### 1.url

    PUT /user/image

##### 2.请求参数

参数|含义|类型
--|--|--
file|头像图片|MultipartFile

##### 3.响应
key：userImg
参数|含义
--|--
0|此功能存在异常，请联系管理员
1|上传成功
2|上传失败

Json示例
​    
​    {
​        "userImg":0
​    }

### 2.用户查询头像展示

##### 1.url

    GET /user/image

##### 2.请求参数

参数|含义|类型
--|--|--
userId|用户学号|String

Json示例

    {
        “userId”:"2017211005"
    }

##### 3.响应
Json示例

    {
        “userP”:"www.baidu.com"
    }

### 2.当前登陆用户头像

##### 1.url

    GET /session/user/image

##### 2.响应
Json示例

    {
        "userPhoto":"www.baidu.com"
    }
